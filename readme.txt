Widget permettant de voir les actualités de civblogs.akendewa.net

== Description ==

Tabs CivBlogs est un widget Wordpress permettant d'afficher les articles récents de civblogs, le top 5 des articles ainsi que les meilleurs tags postés sur civblogs.akendewa.net

== Installation ==

1. Telecharger et dezipper le fichier `Tabscivblogs.zip` dans le dossier `/wp-content/plugins/`
2. Activer le plugin depuis le panneau d'administration (Menu Extension).
3. Glisser deposer le plugin depuis les widgets disponibles vers le MainSidebar
